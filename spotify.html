Why did I undertake this project?

I've often wanted to create a playlist without having to undetake the tedious task of adding individual songs to the playlist. As such, I created a programme that automates this process!

How does it work?

The programme will ask you for the year of music you would like to create a playlist around. It will then find the top 100 songs from that year and add it to a playlist. The user does not have to do anything other than specify which year of music they'd like to listen to.

Technicals

To complete this project I used Python and more importantly the Beautiful Soup library in Python. I used Beautiful soup in order to scrape my desired information from the Billboard 100 websites. I then stored the song names in a list. In order to create the playlist, I had to use the spotify API in order to push a request to create a playlist containing all the songs in my list.
