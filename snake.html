Why did I undertake this project?

I think it's highly likely that most people start their programming journey wanting to know how to create games. This is exactly why I first became interested in programming. When I had developed enough tools, I decided that I wanted to recreate the classic snake game!

How does it work?

The aim of the game is to eat as many of the balls as you can without touching your own tail or the wall! However, everytime you eat a ball, the snake will grow making the difficulty increase!

Technicals

This project utilised a lot of Object Oriented Programming Concepts. I created 3 different objects to recreate this game. These were the snake object, the scoreboard object and the food object. Apart from creating objects, I also had to solve a lot of problems such as knowing when the snake has collided with its own tail, how to make the snake grow and how to randomly regenerate the ball everytime. This project also taught me a lot about the Python 'turtle module' and working with 'GUIs'
