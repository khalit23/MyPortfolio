Why did I undertake this project?

After succesfully building snake, I wanted to try and build another classic game using all the skills that I had learnt.

How does it work?

This is a two player game. The aim is to not let the ball go behind your player or you concede a goal. First person to 5 goals wins! Also, the speed of the ball increases after every hit!

Technicals

I used my Object Oriented Programming skills to create three objects. The player, the ball and the scoreboard. I then used the turtle module to build the screen and to also build the players. I also used inbuilt functions from the turtle module such as 'screen listen' to detect when the user has clicked a key.
